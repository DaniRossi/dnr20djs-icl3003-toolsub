﻿How to start the script:

Method 1:

In Ubuntu 20.04, using the inbuilt file browser open the folder containing index.sh, right click in the empty space and click run in terminal, inside the terminal write “sh ./index.sh” and press enter, this will start the script.

Method 2:

Open the terminal, start the line with “sh” enter the scripts location (e.g. ./home/user/documents/index.sh ), once the command is entered the script should start.
